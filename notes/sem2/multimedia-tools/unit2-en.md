### Step 6: Image Formation

Digital values pixels ke form me organize ki jati hain aur final digital image create hoti hai.

# Q3. Describe the Important Image Attributes such as Image Size, Resolution, Color Depth, Pixel Depth, and Aspect Ratio. Explain Their Impact on Image Quality and Storage Requirements.

Images multimedia systems ka ek essential component hain. Image processing aur multimedia development me kuch important image attributes hote hain jaise Image Size, Resolution, Color Depth, Pixel Depth aur Aspect Ratio. Ye sabhi attributes image ke visual quality aur file size ko determine karte hain.

## What are Image Attributes?

Image Attributes wo characteristics hoti hain jo kisi digital image ki properties ko define karti hain. Ye attributes image ke dimensions, color information, detail level aur display quality ko control karti hain.

Image attributes ka proper selection multimedia project ki performance aur visual quality dono ko affect karta hai.

For example, ek high-resolution image better quality provide karegi lekin uska file size bhi zyada hoga. Isi tarah low-resolution image less storage use karegi lekin image quality compromise ho sakti hai.

## 1. Image Size

Image Size digital image ke dimensions ko represent karta hai. Ye image ki width aur height ko define karta hai.

Image size generally pixels me measure kiya jata hai.

Example:

- 800 × 600 Pixels
- 1024 × 768 Pixels
- 1920 × 1080 Pixels
- 3840 × 2160 Pixels

Yahan first value image ki width aur second value image ki height ko represent karti hai.

### Example

Agar ek image ka size 1920 × 1080 pixels hai, iska matlab image me total:

1920 × 1080 = 2,073,600 Pixels

available hain.

### Impact on Image Quality

Large image size generally more detail provide karta hai.

Bigger dimensions ka matlab hai image me zyada pixels available honge, jisse image clearer aur sharper dikhai degi.

### Impact on Storage Requirements

Image size increase hone par total pixels bhi increase hote hain, jis wajah se storage requirement badh jati hai.

For example:

- Small image = Less Storage
- Large image = More Storage

Multimedia systems me image size aur storage ke beech proper balance maintain karna zaruri hota hai.

## 2. Resolution

Resolution image ke detail level ko represent karta hai. Ye batata hai ki image me kitne pixels available hain.

Resolution ko generally Pixels Per Inch (PPI) ya Dots Per Inch (DPI) me measure kiya jata hai.

### Common Resolution Values

- 72 PPI (Web Images)
- 150 PPI (Medium Quality Printing)
- 300 PPI (High Quality Printing)
- 600 PPI (Professional Printing)

### Example

Agar do images ka size same ho lekin resolution different ho, to higher resolution image zyada detailed aur sharp dikhai degi.

### Impact on Image Quality

High Resolution:

- Better Detail
- Sharper Appearance
- Improved Print Quality

Low Resolution:

- Less Detail
- Blurred Appearance
- Poor Print Quality

### Impact on Storage Requirements

High-resolution images me zyada pixel information hoti hai.

Result:

- Larger File Size
- More Storage Requirement

Low-resolution images:

- Smaller File Size
- Less Storage Requirement

Multimedia developers ko application requirements ke according appropriate resolution choose karni chahiye.

# Q4. Compare BMP, PIC, JPG, PNG, and TIF Image File Formats in Terms of Structure, Compression Techniques, Advantages, Limitations, and Applications.

## Introduction

Digital images multimedia systems ka ek important component hain. Multimedia applications jaise websites, mobile apps, presentations, digital publishing, e-learning systems aur graphic designing me images ka extensive use hota hai. Jab images ko computer me store kiya jata hai, tab unhe kisi specific file format me save kiya jata hai. Image file format image data ko organize aur store karne ka standard method provide karta hai.

Different image formats different purposes ke liye design kiye gaye hain. Kuch formats high-quality image storage ke liye suitable hote hain, kuch web applications ke liye, aur kuch professional printing aur publishing ke liye. Har image format ki apni structure, compression technique, advantages aur limitations hoti hain.

Multimedia development me BMP, PIC, JPG (JPEG), PNG aur TIF (TIFF) sabse commonly used image formats me se hain. In formats ko samajhna multimedia professionals ke liye important hai kyunki image quality, storage requirement aur performance directly image format selection par depend karti hai.

## What is an Image File Format?

Image File Format ek standard structure hota hai jo digital image ko store karne ke liye use kiya jata hai. Ye define karta hai ki image data, color information, metadata aur compression information file ke andar kaise store hogi.

Image format ka selection image quality, storage size aur application requirements ke according kiya jata hai.

## BMP (Bitmap Image Format)

BMP ka full form **Bitmap Image File** hai. Ye Microsoft dwara develop kiya gaya ek raster graphics format hai.

BMP format image ko pixel-by-pixel store karta hai aur generally uncompressed image storage provide karta hai.

### Structure of BMP

BMP file me mainly following components hote hain:

- File Header
- Bitmap Header
- Color Information
- Pixel Data

Har pixel ki information directly file me store hoti hai.

### Compression Technique

BMP usually compression use nahi karta.

Kuch versions Run-Length Encoding (RLE) support karte hain, lekin most BMP files uncompressed hoti hain.

### Advantages of BMP

- High Image Quality
- Simple Structure
- Easy Processing
- No Data Loss

BMP images original image information preserve karti hain.

### Limitations of BMP

- Very Large File Size
- Not Suitable for Web Use
- High Storage Requirement

### Applications of BMP

- Windows Applications
- Image Editing
- Temporary Image Storage
- Graphics Processing

## PIC (Picture File Format)

PIC ek raster image format hai jo different graphics software packages me use kiya jata tha. Ye format early multimedia aur graphic applications me kaafi popular tha.

PIC format multiple versions me available raha hai aur different software vendors ne apne implementations provide kiye hain.

### Structure of PIC

PIC files image data ke saath color information aur image attributes store karti hain.

### Compression Technique

PIC format generally Run-Length Encoding (RLE) compression support karta hai.

Ye repetitive image data ko efficiently compress karta hai.

### Advantages of PIC

- Better Compression Than BMP
- Faster Storage
- Reduced File Size

### Limitations of PIC

- Limited Compatibility
- Less Popular Today
- Not Widely Supported

### Applications of PIC

- Early Multimedia Systems
- Graphic Design Applications
- Legacy Software Systems

Aaj ke time me PIC format ka use kaafi kam ho gaya hai.

## JPG (JPEG)

JPEG ka full form **Joint Photographic Experts Group** hai.

Ye duniya ka sabse popular image format mana jata hai aur digital photography aur web graphics me extensively use hota hai.

JPEG specifically photographic images ke liye design kiya gaya hai.

### Structure of JPEG

JPEG file me image data compressed form me store hota hai.

File structure me generally:

- Header Information
- Quantization Tables
