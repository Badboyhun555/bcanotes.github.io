# Introduction to Multimedia

The word Multimedia is formed using two words:

- Multi → Many
- Media → Medium of communication

Therefore, Multimedia means the use of multiple forms of media together to communicate information more effectively.

In simple words, multimedia is a technology that combines:
- Text
- Images
- Audio
- Video
- Animation

to create attractive and interactive content.

Multimedia helps users understand information more easily because instead of only reading text, users can also:
- Watch videos
- Listen to sound
- View images
- See animations

This makes learning and communication much more interesting.

---

Multimedia is the integration of different types of media such as text, graphics, sound, animation, and video using computer systems for communication and interaction.

# Object Linking and Embedding (OLE)

OLE stands for Object Linking and Embedding.

OLE is a Microsoft technology that allows inserting objects from one application into another.

---

# Example of OLE

Examples:
- Excel chart inside Word document
- PowerPoint slide inside Word file

---

# Types of OLE

1. Linking
2. Embedding

---

# Linking

Linked object remains connected to original file.

If original file changes:
- Linked object updates automatically

---

# Embedding

Embedded object becomes part of destination file.

Changes in original file do not affect embedded object.

---

# Difference Between Linking and Embedding

| Linking | Embedding |
|---|---|
| Connected to original file | Independent copy |
| Updates automatically | No automatic update |
| Smaller file size | Larger file size |

---

# Advantages of OLE

- Better integration
- Easy sharing
- Dynamic updates
- Improved productivity
